# 👨‍💼 Portfólio Profissional

Portfólio pessoal de Johnny dos Anjos Galdino Santana - Desenvolvedor Front-end e Fullstack

## 📝 Descrição

Este é um portfólio profissional desenvolvido com HTML e CSS puro. O projeto apresenta informações sobre experiência profissional, links para redes sociais, currículo e referências a outras páginas do site.

## 🎯 Objetivo

Servir como página principal de apresentação profissional, com acesso fácil aos principais canais de contato e informações sobre a trajetória profissional.

## 📁 Estrutura do Projeto

```
Perfil_Profissional_2/
├── index.html          # Página principal do portfólio
├── style.css           # Estilos e layout
├── img/
│   ├── perfil2.jpg    # Foto de perfil
│   └── img2.jpg       # Imagem de fundo
├── sobre.html         # Página sobre mim (referenciada)
├── projetos.html      # Página de projetos (referenciada)
└── README.md          # Este arquivo
```

## 🚀 Como Usar

1. **Abrir o projeto**: Abra o arquivo `index.html` em seu navegador
2. **Navegação**: Use os links do menu para acessar:
   - 📄 Currículo (PDF externo)
   - 💼 LinkedIn
   - 🐙 GitHub
   - 💬 WhatsApp
   - 📖 Sobre Mim
   - 💻 Projetos

## 🎨 Características

- ✅ Design responsivo e centrado
- ✅ Foto de perfil circular
- ✅ Menu de navegação intuitivo
- ✅ Efeitos hover nos botões
- ✅ Fundo com imagem personalizada
- ✅ Layout limpo e profissional
- ✅ Cores harmoniosas (azul e roxo)

## 💻 Tecnologias Utilizadas

- **HTML5** - Estrutura e semântica
- **CSS3** - Estilos, layout e animações
  - Flexbox/Grid concepts
  - Transições suaves
  - Border-radius e efeitos visuais

## 🎨 Paleta de Cores

- **Azul do container**: #89afd2
- **Roxo dos botões**: #5f5fdd
- **Roxo hover**: #525286
- **Fundo**: Imagem personalizada com cor cinza (#f0f0f0)

O projeto apresenta um container com largura fixa (500px) centralizado na página, adequado para visualização em diferentes tamanhos de tela.
